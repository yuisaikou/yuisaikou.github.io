内联示例 后面文字不换行
也可以只传入一个参数，文字写在后边（这样不支持外联）
复选框：
{% cb text, checked?, incline? %}
text：显示的文字
checked：默认是否已勾选，默认 false
incline: 是否内联（可以理解为后面的文字是否换行），默认 false
-----------------------------------------------------

{% btn url, text, title %}

n1-n2-...：每行的图片数量，可以省略，默认单行最多 3 张图，求和必须相等于 total，否则按默认样式
